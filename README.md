# anonim-chat
Telegram Bot for chat with random stranger anonymously

## Run project localy
- ```git clone https://github.com/Shiyinq/anonim-chat.git```
- ```cd anonim-chat```
- ```npm install```
- rename ```.env.example``` to ```.env``` and edit ```.env``` with your own bot token and mongo uri
- ```npm run dev```
